## Prime Cars (Next.js)

### Run locally

```bash
cd "/Users/vahagn/Documents/other/prime-cars"

# install dependencies
npm install

# start dev server

npm run dev
```

Then open the URL printed in the terminal (typically `http://localhost:3000`).

### Routes

- `/` Home
- `/cars`
- `/calculator`
- `/partners`
- `/about`
