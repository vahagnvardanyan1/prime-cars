# Tools and Libraries Used in This Project

This document provides an overview of the main tools, libraries, and technologies used in this project.

## Framework
- **Next.js**: A React framework for building server-side rendering and static web applications.

## React Libraries
- **React**: JavaScript library for building user interfaces.
- **React DOM**: Serves as the entry point to the DOM and server renderers for React.
- **react-hook-form**: For managing form state and validation.
- **formik**: Another popular library for form handling.
- **zod**: TypeScript-first schema declaration and validation library.
- **zod-formik-adapter**: Adapter to integrate Zod with Formik.

## UI Component Libraries
- **@radix-ui/react**: Primitives for building accessible React components (accordion, dialog, popover, tooltip, tabs, etc.).
- **cmdk**: Command menu for React.
- **embla-carousel-react**: Carousel library.
- **lucide-react**: Icon library.

## State & Data Fetching
- **@tanstack/react-query**: Powerful asynchronous state management for React.
- **@tanstack/react-query-devtools**: Devtools for React Query.

## Utilities
- **clsx**: Utility for constructing `className` strings conditionally.
- **class-variance-authority**: Variance management utility for class names.
- **tailwind-merge**: Utility for merging Tailwind CSS classes.
- **jszip**: Library for creating, reading and editing .zip files.

## Styling
- **Tailwind CSS**: Utility-first CSS framework.
- **postcss**: Tool for transforming CSS with JavaScript.
- **autoprefixer**: PostCSS plugin to parse CSS and add vendor prefixes.

## Localization
- **next-intl**: Internationalization for Next.js apps.

## Theming
- **next-themes**: Easy theme switching for Next.js apps.

## Other UI Tools
- **input-otp**: OTP input component.
- **react-day-picker**: Date picker for React.
- **react-resizable-panels**: Resizable panels component.
- **recharts**: Charting library built with React components.
- **sonner**: Toast notification library.
- **vaul**: State management utility.

## Development Tools
- **TypeScript**: Typed superset of JavaScript.
- **ESLint**: Linting tool for JavaScript and TypeScript.
- **eslint-config-next**: ESLint configuration for Next.js.

## Node.js Types
- **@types/node**
- **@types/react**
- **@types/react-dom**


This document can be updated as more tools or libraries are added or removed from the project.
